Python 教材（為資深 Java 工程師設計）

以 Java ↔ Python 對照 切入，重點放在思維轉換（mindset shift）與 Pythonic 寫法。 完整課程規劃見 `curriculum.md`；本書範例以 Python 3.12+ 為準。

心法：不要把 Java 的寫法直接翻譯成 Python——先學會「Python 怎麼想」， 再利用你的工程經驗（測試、架構、型別）讓 Python 程式碼比一般人寫得更穩。

第一部分：快速上手與思維轉換

  • 第 1 章：Python 生態系與開發環境 — CPython / bytecode / PVM 對照 JVM、venv 與 uv、pyproject.toml、ruff 與 mypy、REPL 互動式開發。
  • 第 2 章：語法速覽——從 Java 翻譯到 Python — 縮排即區塊、name binding、一切皆物件（int 不溢位）、is vs ==、None、truthiness。
  • 第 3 章：函式——Python 的一等公民 — 函式即物件、*args/**kwargs、可變預設參數陷阱、closure 與 nonlocal、type hints、docstring 與 PEP 257（對照 JavaDoc）。

第二部分：資料結構與 Pythonic 寫法

  • 第 4 章：核心資料結構深入 — list/tuple/dict/set 內部實作與複雜度、可變性、切片、解包、collections、深淺拷貝。
  • 第 5 章：Pythonic 慣用法 — comprehension 對照 Stream、惰性求值、enumerate/zip/sorted、EAFP、duck typing、with、海象運算子與 f-string。
  • 第 6 章：迭代器與產生器 — 迭代協定、yield 與 generator、itertools、generator pipeline、yield from。

第三部分：物件導向與型別系統

  • 第 7 章：Python 的類別與物件 — class/__init__/self、name mangling、類別 vs 實例屬性、@property、@classmethod/@staticmethod、@dataclass、__repr__/__eq__/__hash__。
  • 第 8 章：繼承、協定與魔術方法 — super()、多重繼承與 MRO、abc.ABC、Protocol（結構型別）vs ABC（名義型別）、魔術方法、mixin、組合優於繼承。
  • 第 9 章：型別提示與靜態檢查 — 現代 typing 寫法、泛型（3.12 [T] 語法）與型別抹除、Protocol、mypy/pyright 與 CI、pydantic 執行期驗證、漸進式型別。
  • 第 10 章：裝飾器與元程式設計 — 裝飾器與 functools.wraps、帶參數裝飾器、類別裝飾器、描述器（descriptor）、__getattr__/__getattribute__、metaclass 與 __init_subclass__。

第四部分：工程實務

  • 第 11 章：模組、套件與專案結構 — import 機制與 sys.path、套件與 __init__.py、__main__.py、src layout、pyproject.toml 與 entry points、uv 相依管理。
  • 第 12 章：例外處理與防禦性程式設計 — 例外階層、EAFP、try/except/else/finally、自訂例外、例外鏈（raise from）、contextlib、防禦性設計與 logging。
  • 第 13 章：測試 — pytest、assert rewriting、fixture 與 conftest.py、參數化、unittest.mock 與 @patch、coverage、property-based testing（hypothesis）。
  • 第 14 章：並行與非同步 — GIL 與 free-threading（PEP 703/779）、threading vs multiprocessing、concurrent.futures、async/await 與函式著色、asyncio、Queue。
  • 第 15 章：檔案、I/O 與標準函式庫巡禮 — pathlib、文字/二進位 I/O 與編碼、json/csv、datetime、tempfile、subprocess、常用標準函式庫地圖。

第五部分：實戰應用（依興趣選修）

  • 第 16 章：Web 開發 — FastAPI（型別驅動驗證 + 自動 OpenAPI）對照 Spring Boot、Depends 函式式 DI 與 dependency_overrides、SQLAlchemy 2.0（Mapped/mapped_column/select、identity map）對照 JPA/Hibernate、ASGI vs WSGI、uvicorn/gunicorn worker 與 Docker 部署。
  • 第 17 章：資料處理與自動化腳本 — requests vs httpx（連線池與 async）、pandas DataFrame 與向量化思維、BeautifulSoup（本地 HTML 可重現解析）對照 Jsoup、openpyxl 報表自動化對照 Apache POI、排程與系統腳本取代 shell script。
  • 第 18 章：總整專案（Capstone） — src layout 分層（API/service/持久層）、型別三分（pydantic 邊界／SQLAlchemy 持久／dataclass 內部）、FastAPI + click 雙入口共用 service、pytest + TestClient + in-memory DB 測試、ruff + mypy + coverage 全綠、GitHub Actions CI、uv build 打包。

閱讀方式

每章獨立成檔，結構一致：

  1. 引言——從 Java 工程師視角點出本章會顛覆的直覺。
  2. Java ↔ Python 三段式對照——「Java 怎麼做 → Python 怎麼做 → 思維差異」。
  3. 陷阱框（⚠️）／Pythonic 重點框（💡）／Java 對照框（🔁）。
  4. 本章小結 + 練習。

所有 Python 範例皆經 python3 實際執行驗證，型別提示範例經 mypy 檢查。
